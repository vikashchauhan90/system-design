### 55. Detect Cycle in Linked List

#### 55.1 [Brute Force - Hash Set]

**Pseudocode:**
```
FUNCTION DetectCycleHashSet(head)
  visited ← EMPTY_SET
  current ← head

  WHILE current != NULL DO
    IF current IN visited THEN
      RETURN TRUE  // Cycle found
    END IF
    visited.ADD(current)
    current ← current.next
  END WHILE

  RETURN FALSE  // No cycle
END FUNCTION
```

**Code Implementation:**
```csharp
public static bool DetectCycleHashSet(ListNode head)
{
    var visited = new HashSet<ListNode>();
    ListNode current = head;

    while (current != null)
    {
        if (visited.Contains(current))
            return true;

        visited.Add(current);
        current = current.next;
    }

    return false;
}
```

- **Time Complexity :** `O(n)`
- **Space Complexity :** `O(n)`

#### 55.2 [Two-Pointer (Floyd's Cycle Detection)]

**Pseudocode:**
```
FUNCTION DetectCycleFloyd(head)
  slow ← head
  fast ← head

  WHILE fast != NULL AND fast.next != NULL DO
    slow ← slow.next         // Move 1 step
    fast ← fast.next.next    // Move 2 steps

    IF slow == fast THEN      // Cycle detected
      RETURN TRUE
    END IF
  END WHILE

  RETURN FALSE
END FUNCTION
```

**Code Implementation:**
```csharp
public static bool DetectCycleFloyd(ListNode head)
{
    ListNode slow = head;
    ListNode fast = head;

    while (fast != null && fast.next != null)
    {
        slow = slow.next;
        fast = fast.next.next;

        if (slow == fast)
            return true;
    }

    return false;
}
```

- **Time Complexity :** `O(n)`
- **Space Complexity :** `O(1)`
