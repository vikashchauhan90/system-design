## First Unique Character in a String
Given a string s, find the first non-repeating character in it and return its index. If the string does not contain a unique character, return -1.

```
Examples:
- s = "leetcode" → 0 (l)
- s = "loveleetcode" → 2 (v)
- s = "aabb" → -1
```

### [Brute Force - Nested Loop]

```csharp
public static int FirstUniqCharBruteForce(string s)
{
    // For each character, check if it appears elsewhere
    for (int i = 0; i < s.Length; i++)
    {
        bool isUnique = true;
        for (int j = 0; j < s.Length; j++)
        {
            if (i != j && s[i] == s[j])
            {
                isUnique = false;
                break;
            }
        }

        if (isUnique)
        {
            return i;
        }
    }

    return -1;
}
```

```text
s = "leetcode"

i=0 (l): Check against all → appears once → return 0 ✓

Result: 0
```

- **Time Complexity:** `O(n²)` - nested loops
- **Space Complexity:** `O(1)` - no extra space

#### 44.2 [HashMap - O(n)]

```csharp
public static int FirstUniqCharHashMap(string s)
{
    // Count frequency of each character
    Dictionary<char, int> freq = new();
    foreach (char c in s)
    {
        if (freq.ContainsKey(c))
            freq[c]++;
        else
            freq[c] = 1;
    }

    // Find first character with frequency 1
    for (int i = 0; i < s.Length; i++)
    {
        if (freq[s[i]] == 1)
        {
            return i;
        }
    }

    return -1;
}
```

```text
s = "leetcode"

Pass 1 - Count:
l:1, e:3, t:1, c:1, o:1, d:1

Pass 2 - Find First with count=1:
i=0: l has count 1 → return 0 ✓

Result: 0
```

- **Time Complexity:** `O(n)` - two passes
- **Space Complexity:** `O(k)` - frequency map of alphabet size
