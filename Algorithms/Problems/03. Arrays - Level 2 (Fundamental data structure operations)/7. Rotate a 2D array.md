## Rotate 2D Array (Matrix Rotation)

**Description:**
Rotate a given **N × N 2D matrix** by **90 degrees clockwise**.

Matrix rotation means transforming rows into columns while maintaining relative ordering.

---

## Examples:

```
Input:
[
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

Output (90° Clockwise):
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3]
]
```

```
Input:
[
  [1, 2],
  [3, 4]
]

Output:
[
  [3, 1],
  [4, 2]
]
```

```
Input:
[
  [5]
]

Output:
[
  [5]
]
```

---

## Constraints

* Matrix is square: `N × N`
* `1 ≤ N ≤ 10^3`
* In-place rotation preferred
* No extra matrix unless specified
* Elements are within integer range

---

# ✅ Approach 1: Using Extra Matrix (Brute Force)

We create a new matrix and place elements in rotated positions.

**Key Idea:**

For 90° clockwise rotation:

```
newMatrix[j][n - 1 - i] = matrix[i][j]
```

---

### Pseudocode:

```
FUNCTION RotateMatrix(matrix):
    n = length of matrix
    CREATE newMatrix of size n x n

    FOR i = 0 TO n-1:
        FOR j = 0 TO n-1:
            newMatrix[j][n-1-i] = matrix[i][j]

    RETURN newMatrix
END FUNCTION
```

---

### Code Implementation (C#):

```csharp
public static int[,] RotateMatrix(int[,] matrix)
{
    int n = matrix.GetLength(0);
    int[,] result = new int[n, n];

    for (int i = 0; i < n; i++)
    {
        for (int j = 0; j < n; j++)
        {
            result[j, n - 1 - i] = matrix[i, j];
        }
    }

    return result;
}
```

---

### Visualization:

Original:

```
1 2 3
4 5 6
7 8 9
```

Mapping:

```
(0,0) → (0,2)
(0,1) → (1,2)
(0,2) → (2,2)
...
```

Result:

```
7 4 1
8 5 2
9 6 3
```

---

### Complexity

* **Time Complexity:** `O(n²)`
* **Space Complexity:** `O(n²)` (extra matrix)

---

# ✅ Approach 2: In-Place Rotation (Transpose + Reverse)

This is the most optimal and commonly asked approach.

### Key Steps:

1. Transpose the matrix
2. Reverse each row

---

## Step 1: Transpose

Swap:

```
matrix[i][j] ↔ matrix[j][i]
```

---

## Step 2: Reverse Each Row

Reverse elements in every row.

---

### Pseudocode:

```
FUNCTION RotateMatrixInPlace(matrix):
    n = length of matrix

    // Step 1: Transpose
    FOR i = 0 TO n-1:
        FOR j = i TO n-1:
            SWAP matrix[i][j] WITH matrix[j][i]

    // Step 2: Reverse each row
    FOR i = 0 TO n-1:
        REVERSE matrix[i]

END FUNCTION
```

---

### Code Implementation:

```csharp
public static void RotateMatrixInPlace(int[,] matrix)
{
    int n = matrix.GetLength(0);

    // Step 1: Transpose
    for (int i = 0; i < n; i++)
    {
        for (int j = i; j < n; j++)
        {
            int temp = matrix[i, j];
            matrix[i, j] = matrix[j, i];
            matrix[j, i] = temp;
        }
    }

    // Step 2: Reverse each row
    for (int i = 0; i < n; i++)
    {
        int left = 0;
        int right = n - 1;

        while (left < right)
        {
            int temp = matrix[i, left];
            matrix[i, left] = matrix[i, right];
            matrix[i, right] = temp;

            left++;
            right--;
        }
    }
}
```

---

### Visualization

### Original:

```
1 2 3
4 5 6
7 8 9
```

### After Transpose:

```
1 4 7
2 5 8
3 6 9
```

### After Row Reverse:

```
7 4 1
8 5 2
9 6 3
```

---

### Complexity

* **Time Complexity:** `O(n²)`
* **Space Complexity:** `O(1)` (in-place)

# ✅ Approach 3: Layer-by-Layer Rotation (Four-Way Swap)

Rotate elements in groups of four.

### Idea:

Move elements in circular manner:

```
top → right
right → bottom
bottom → left
left → top
```

---

### Pseudocode:

```
FUNCTION RotateMatrixLayer(matrix):
    n = length of matrix

    FOR layer = 0 TO n/2:
        first = layer
        last = n - 1 - layer

        FOR i = first TO last-1:
            offset = i - first

            top = matrix[first][i]

            matrix[first][i] = matrix[last-offset][first]
            matrix[last-offset][first] = matrix[last][last-offset]
            matrix[last][last-offset] = matrix[i][last]
            matrix[i][last] = top
END FUNCTION
```

---

### Code Implementation:

```csharp
public static void RotateMatrixLayer(int[,] matrix)
{
    int n = matrix.GetLength(0);

    for (int layer = 0; layer < n / 2; layer++)
    {
        int first = layer;
        int last = n - 1 - layer;

        for (int i = first; i < last; i++)
        {
            int offset = i - first;

            int top = matrix[first, i];

            // left -> top
            matrix[first, i] = matrix[last - offset, first];

            // bottom -> left
            matrix[last - offset, first] = matrix[last, last - offset];

            // right -> bottom
            matrix[last, last - offset] = matrix[i, last];

            // top -> right
            matrix[i, last] = top;
        }
    }
}
```

---

### Visualization (First Layer Only)

```
1 2 3
4 5 6
7 8 9
```

Swap 4 elements at a time:

```
1 → 3 → 9 → 7 → 1
2 → 6 → 8 → 4 → 2
```

Final:

```
7 4 1
8 5 2
9 6 3
```

---

### Complexity

* **Time Complexity:** `O(n²)`
* **Space Complexity:** `O(1)`
