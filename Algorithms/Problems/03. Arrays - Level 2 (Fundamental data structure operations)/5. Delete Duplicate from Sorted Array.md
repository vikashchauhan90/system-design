## Delete Duplicate from Sorted Array

**Description:** Given a sorted array `nums`, remove the duplicates in-place such that each element appears only once and return the new length. Do not allocate extra space for another array — you must do this by modifying the input array in-place with O(1) extra memory.

After removing the duplicates, the first `k` elements of `nums` should hold the unique elements in the original relative order. The function should return `k` (the number of unique elements).

**Examples:**
```
Input: nums = [1,1,2]
Output: k = 2, nums = [1,2,...]
Explanation: After removing duplicates, the first two elements are 1 and 2.

Input: nums = [0,0,1,1,1,2,2,3,3,4]
Output: k = 5, nums = [0,1,2,3,4,...]
Explanation: Unique elements are 0,1,2,3,4 in order.

Input: nums = []
Output: k = 0
```

#### Constraints
- `nums` is sorted in non-decreasing order.
- `0 <= nums.length <= 10^5` (or similar bounds depending on context).
- The algorithm must use O(1) extra space and run in O(n) time.

### Approach (Two Pointers — In-place)

Use a slow pointer `i` to mark the position of the last unique element and a fast pointer `j` to scan the array. Whenever `nums[j]` differs from `nums[i]`, increment `i` and set `nums[i] = nums[j]`. At the end, the length of unique elements is `i + 1` (or `0` if array is empty).

This preserves relative order and performs writes only when a new unique value is found.

### Pseudocode

```
FUNCTION RemoveDuplicates(nums):
    IF nums IS NULL OR LENGTH(nums) == 0: RETURN 0

    // slow points to last unique element index
    i = 0
    FOR j FROM 1 TO LENGTH(nums)-1:
        IF nums[j] != nums[i]:
            i = i + 1
            nums[i] = nums[j]
        END IF
    END FOR

    RETURN i + 1
END FUNCTION
```

```csharp
// C# implementation (in-place)
public static int RemoveDuplicates(int[] nums)
{
    if (nums == null || nums.Length == 0) return 0;

    int i = 0; // last unique index
    for (int j = 1; j < nums.Length; j++)
    {
        if (nums[j] != nums[i])
        {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1; // length of unique prefix
}
```

### Visualization (step-by-step)

Example: `nums = [0,0,1,1,1,2,2,3,3,4]`
- Start: i=0 (nums[0]=0)
- j=1: nums[1]=0 equals nums[i]=0 → skip
- j=2: nums[2]=1 != nums[i]=0 → i=1; nums[1]=1 → array becomes [0,1,1,1,1,2,2,3,3,4]
- j=3: nums[3]=1 == nums[i]=1 → skip
- j=4: skip
- j=5: nums[5]=2 != nums[i]=1 → i=2; nums[2]=2 → [0,1,2,1,1,2,2,3,3,4]
- continue similarly → final first k=5 elements: [0,1,2,3,4]

Visual state transitions: [0,0,1,...] -> [0,1,1,...] -> [0,1,2,...] -> ... -> [0,1,2,3,4,...]

### Complexity
- **Time Complexity :** `O(n)` — single pass through the array.
- **Space Complexity :** `O(1)` — only constant extra variables used (in-place modifications).
