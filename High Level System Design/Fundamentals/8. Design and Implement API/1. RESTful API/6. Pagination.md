
 
# Pagination, Filtering, Sorting

### Introduction
Most API endpoints returning collections need pagination to limit response size, plus filtering and sorting to let clients retrieve only what they need.

### Common Query Parameters

| Parameter | Example | Description |
|-----------|---------|-------------|
| `page` | `?page=2` | Page number (1‑based). |
| `limit` (or `size`) | `?limit=20` | Items per page (default 20, max 100). |
| `sort` | `?sort=name,-date` | Field name; prefix `-` for descending. |
| `filter` | `?price_gte=100` | Custom filter syntax (e.g., `field_op=value`). |
| `fields` | `?fields=id,name` | Projection – return only specific fields. |

### Best Practices
- Use **1‑based pagination** (page=1 returns first page).
- Provide metadata in response: `totalCount`, `page`, `limit`, `next`/`prev` links.
- Use **cursor‑based pagination** for very large, frequently changing datasets (uses `after`/`before` tokens).
- Validate and sanitize sorting/filtering parameters to avoid injection.
- Document your filter syntax (e.g., `eq`, `gt`, `lt`, `like`).

### C# Implementation

**Request Model**
```csharp
public class QueryParameters
{
    private const int MaxLimit = 100;
    private int _limit = 20;
    public int Page { get; set; } = 1;
    public int Limit
    {
        get => _limit;
        set => _limit = (value > MaxLimit) ? MaxLimit : value;
    }
    public string Sort { get; set; } = "id";
    public string? Filter { get; set; }
}
```

**Paged Response Model**
```csharp
public class PagedResponse<T>
{
    public List<T> Data { get; set; } = new();
    public int Page { get; set; }
    public int Limit { get; set; }
    public int TotalCount { get; set; }
    public int TotalPages => (int)Math.Ceiling((double)TotalCount / Limit);
    public bool HasNext => Page < TotalPages;
    public bool HasPrev => Page > 1;
    public string? NextLink => HasNext ? $"?page={Page+1}&limit={Limit}" : null;
    public string? PrevLink => HasPrev ? $"?page={Page-1}&limit={Limit}" : null;
}
```

```csharp
[ApiController]
[Route("api/products")]
public class ProductsController : ControllerBase
{
    private readonly IRepository<Product> _repository;

    [HttpGet]
    public async Task<ActionResult<PagedResponse<Product>>> GetAll([FromQuery] QueryParameters query)
    {
        // Apply filtering (simplified)
        var queryable = _repository.Query();
        if (!string.IsNullOrEmpty(query.Filter))
        {
            // Parse filter expression, e.g., "price>100"
            // Use a library like Dynamic LINQ or manual parsing
        }

        // Apply sorting
        queryable = ApplySorting(queryable, query.Sort);

        // Get total count before pagination
        var totalCount = await queryable.CountAsync();

        // Apply pagination
        var data = await queryable
            .Skip((query.Page - 1) * query.Limit)
            .Take(query.Limit)
            .ToListAsync();

        var response = new PagedResponse<Product>
        {
            Data = data,
            Page = query.Page,
            Limit = query.Limit,
            TotalCount = totalCount
        };

        // Add HATEOAS links (optional)
        Response.Headers.Add("X-Total-Count", totalCount.ToString());

        return Ok(response);
    }

    private IQueryable<Product> ApplySorting(IQueryable<Product> query, string sort)
    {
        // "name,-price" => order by name ascending, then price descending
        var sortParams = sort.Split(',');
        IOrderedQueryable<Product>? orderedQuery = null;
        for (int i = 0; i < sortParams.Length; i++)
        {
            var param = sortParams[i];
            var descending = param.StartsWith('-');
            var property = descending ? param[1..] : param;

            if (i == 0)
                orderedQuery = descending ? query.OrderByDescending(p => EF.Property<object>(p, property)) : query.OrderBy(p => EF.Property<object>(p, property));
            else
                orderedQuery = descending ? orderedQuery.ThenByDescending(p => EF.Property<object>(p, property)) : orderedQuery.ThenBy(p => EF.Property<object>(p, property));
        }
        return orderedQuery ?? query;
    }
}
```
