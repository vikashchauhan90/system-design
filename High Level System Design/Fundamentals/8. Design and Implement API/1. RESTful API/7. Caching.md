

# Caching (ETag, Last-Modified)

### Introduction
HTTP caching reduces server load and latency by allowing clients (and intermediaries) to reuse previously fetched responses. Two main headers:
- **ETag** – opaque identifier (hash) of the resource representation.
- **Last-Modified** – timestamp of last change.

Clients send `If-None-Match` (with ETag) or `If-Modified-Since` (with Last-Modified). Server returns `304 Not Modified` if the resource hasn’t changed.

### Best Practices
- Use **strong ETags** (enclosed in double quotes, e.g., `"abc123"`) for byte‑exact representations.
- For dynamic resources, generate ETag from a hash of the content or a version number.
- Always include `Cache-Control` headers to control caching behaviour (`public`, `private`, `max-age`).
- Do not cache sensitive (user‑specific) data unless using `private`.

### C# Implementation

**ETag Generation Helper**
```csharp
public static class ETagHelper
{
    public static string GenerateETag(byte[] contentBytes)
    {
        using var sha = SHA256.Create();
        var hash = sha.ComputeHash(contentBytes);
        return $"\"{Convert.ToBase64String(hash)}\"";
    }
}
```

**Controller with ETag and Last-Modified**
```csharp
[ApiController]
[Route("api/products")]
public class ProductsController : ControllerBase
{
    private readonly IRepository<Product> _repository;

    [HttpGet("{id}")]
    public async Task<IActionResult> GetProduct(int id)
    {
        var product = await _repository.GetByIdAsync(id);
        if (product == null) return NotFound();

        // Generate ETag from JSON representation
        var jsonBytes = JsonSerializer.SerializeToUtf8Bytes(product);
        var etag = ETagHelper.GenerateETag(jsonBytes);

        // Check If-None-Match header
        if (Request.Headers.TryGetValue("If-None-Match", out var incomingEtag) && incomingEtag == etag)
            return StatusCode(304);

        // Check Last-Modified (if product has UpdatedAt timestamp)
        var lastModified = product.UpdatedAt ?? product.CreatedAt;
        if (Request.Headers.TryGetValue("If-Modified-Since", out var modifiedSince) &&
            DateTimeOffset.TryParse(modifiedSince, out var since) &&
            lastModified <= since.UtcDateTime)
        {
            return StatusCode(304);
        }

        // Set response headers
        Response.Headers.ETag = etag;
        Response.Headers.LastModified = lastModified.ToString("R");
        Response.Headers.CacheControl = "public, max-age=3600"; // cache for 1 hour

        return Ok(product);
    }
}
```

**Conditional Update (Optimistic Concurrency)**
```csharp
[HttpPut("{id}")]
public async Task<IActionResult> UpdateProduct(int id, Product updated, [FromHeader(Name = "If-Match")] string ifMatch)
{
    var existing = await _repository.GetByIdAsync(id);
    if (existing == null) return NotFound();

    var currentETag = ETagHelper.GenerateETag(JsonSerializer.SerializeToUtf8Bytes(existing));
    if (ifMatch != currentETag)
        return StatusCode(412, "Precondition failed: resource changed since last fetch.");

    // Proceed with update
    await _repository.UpdateAsync(updated);
    return NoContent();
}
```

**Response Headers Example**
```
HTTP/1.1 200 OK
ETag: "dGVzdA=="
Last-Modified: Wed, 02 Apr 2025 10:00:00 GMT
Cache-Control: public, max-age=3600
```

**Client Usage**
```http
GET /api/products/123
If-None-Match: "dGVzdA=="
```

If unchanged, server returns `304 Not Modified` with no body.
